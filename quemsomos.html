Bem-vindo(a)! Meu nome é Gabriel Frameschi e sou estudante de Análise e Desenvolvimento de Sistemas na UniToledo Araçatuba. Essa página é um projeto feito para a matéria de Desenvolvimento Web, aplicada pelo Prof. M.e Fabrício Quintanilha.

O objetivo desse projeto foi aplicar tudo (e mais um pouco) que foi visto durante as aulas, praticando os códigos e fixando a teoria. Graças ao Bootstrap 5, foi possível montar todo o design do site de forma mais rápida (mesmo sendo difícil compreender ele no começo 🤣).

Também foram aplicados vários conceitos por aqui, como o básico de design de interfaces e layouts, com o propósito de oferecer a melhor experiência possível nos mais diferentes tipos de telas. Segue uma pequena lista com as tecnologias e recursos usados:

  • HTML5 e CSS3 (o feijão com arroz)
  • JavaScript: Alguns scripts de funcionalidades do Bootstrap
  • Bootstrap 5: biblioteca de estilos e funções (base de toda a página)
  • Alguns conceitos de UI e UX (Design de Interfaces)

Espero que tenham gostado. ;)
